# htjworld

비행기로 날아다니며 블로그 글을 탐험하는 3D 포트폴리오 사이트.

## 조작법

- `WASD` / 방향키 — 이동
- `Space` — 상승
- `Ctrl` / `C` — 하강
- `Shift` — 부스트
- `클릭` — 빌딩 광고 링크 이동

## 맵

- Creative City
- Ice Prison
- Maze Runner
- Shining Hallway

## 기술 스택

- React + TypeScript
- Three.js / React Three Fiber
- @react-three/drei, @react-three/cannon
- Zustand
- Vite → GitHub Pages 자동 배포
